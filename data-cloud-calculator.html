Pros

  • ✓ Native Salesforce unification - ML-powered identity resolution included
  • ✓ Unified profiles available in SF UI - Relationship managers see 360° view
  • ✓ Batch ingestion is FREE - No DSC cost for SF org ingestion (every 6 hours)
  • ✓ Multi-org sync built-in - Data Cloud One companion orgs included
  • ✓ Faster implementation - 8-12 weeks vs 6+ months for custom Snowflake solution
  • ✓ No custom code maintenance - Salesforce manages deduplication logic

Cons

  • ✗ Companion org sync is expensive - Largest DSC cost component (46% of total)
  • ✗ Output to Snowflake costs DSC - Every sync consumes credits (though batch is cheap)
  • ✗ Profile updates consume DSC - Real-time mode very expensive (use batch!)
  • ✗ Limited transformation in Data Cloud - Complex calculations better in Snowflake
  • ✗ Data latency - 6-hour delay in batch mode (vs real-time in Snowflake)
